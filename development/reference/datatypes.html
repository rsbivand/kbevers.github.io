Data types¶

This section describe the multiplude of data types in use in PROJ.4. As a rule of thumb PROJ.4 data types are prefixed with PJ_, although with a few notable exceptions that can be traced back to the very early days of PROJ.4 when the PJ_ prefix was not consistenly used.

PROJ.4 comes with a large range of coordinate data types, some are simple and generic, others are complex and made for a very specific purpose.

2 dimensional coordinates¶

LP¶

Basic type for a pair of longitude and latitude. The unit of longitude and latitude is usually radians.

typedef struct { double lam, phi; } LP;
double LP.lam¶

Longitude. Lambda.

double LP.phi¶

Latitude. Phi.

XY¶

Basic type for a projected coordinate.

typedef struct { double x, y; } XY;
double XY.lam¶

Easting.

double XY.phi¶

Northing.

UV¶

Basic type for a coordinate. Can be either geodetic or projected.

typedef struct {double u, v; } UV;
double UV.u¶

Longitude or easting, depending on use.

double UV.v¶

Latitude or northing, depending on use.

PJ_EN¶

Generic easting/northing coordinate.

typedef struct { double e, n; } PJ_EN;
double PJ_EN.e¶

Easting

double PJ_EN.n¶

Northing

3 dimensional coordinates¶

LPZ¶

Basic type for a triplet of longitude, latitude and a height. The unit of longitude and latitude is usually radians and meters for the height.

typedef struct { double lam, phi, z; } LP;
double LPZ.lam¶

Longitude. Lambda.

double LPZ.phi¶

Latitude. Phi.

double LPZ.z¶

Vertical component.

XYZ¶

Basic type for a 3-dimensional projected coordinate.

typedef struct { double x, y; } XY;
double XYZ.x¶

Easting.

double XYZ.y¶

Northing.

double XYZ.z¶

Vertical component.

UVW¶

Basic type for a 3-dimensional coordinate. Can be either geodetic or projected.

typedef struct {double u, v, w; } UVW;
double UVW.u¶

Longitude or easting, depending on use.

double UVW.v¶

Latitude or northing, depending on use.

double UVW.w¶

Vertical component.

PJ_ENH¶

Easting, northing and height.

typedef struct {double e, n, h; } PJ_ENH
double PJ_ENH.e¶

Easting

double PJ_ENH.n¶

Northing

double PJ_ENH.h¶

Height

PJ_TRIPLET¶

Union type that groups all coordinate data types of two and tree dimensions.

typedef union PJ_TRIPLET {
    PJ_OPK  opk;
    PJ_ENH  enh;
    PJ_EZN  ezn;
    PJ_DMS  dms;
    double v[3];
    XYZ    xyz;
    LPZ    lpz;
    UVW    uvw;
    XY     xy;
    LP     lp;
    UV     uv;
    PJ_AF  af;
} PJ_TRIPLET;

Spatiotemporal coordinate types¶

The following data types are extensions of the triplets above into the time domain.

PJ_LPZT¶

Spatiotemporal version of LPZ.

typedef struct {
    double lam;
    double phi;
    double z;
    double t;
}
double PJ_LPZT.lam¶

Longitude.

double PJ_LPZT.phi¶

Latitude

double PJ_LPZT.z¶

Vertical component.

double PJ_LPZT.t¶

Time component.

PJ_XYZT¶

Generic spatiotemporal coordinate. Usefull for e.g. cartesian coordinates with an attached time-stamp.

typedef struct {
    double x;
    double y;
    double z;
    double t;
} PJ_XYZT;
double PJ_XYZT.x¶

Easting.

double PJ_XYZT.y¶

Northing.

double PJ_XYZT.z¶

Vertical component.

double PJ_XYZT.t¶

Time component.

PJ_UVWT¶

Spatiotemporal version of PJ_UVW.

typedef struct {double u, v, w, t; } PJ_UVWT
double PJ_UVWT.e¶

First horizontal component.

double PJ_UVWT.n¶

Second horizontal component.

double PJ_UVWT.w¶

Vertical component.

double PJ_UVWT.t¶

Temporal component.

Easting

double PJ_ENHT.n¶

Northing

Height

Ancillary types for geodetic computations¶

PJ_OPK¶

Rotations, for instance three euler angles.

typedef struct { double o, p, k; } PJ_OPK
double PJ_OPK.o¶

First rotation angle, omega.

double PJ_OPK.p¶

Second rotation angle, phi.

double PJ_OPK.k¶

Third rotation angle, kappa.

PJ_DMS¶

Describe a longitude or latitude by degrees, minutes and seconds.

typedef struct { double d, m, s; } PJ_DMS
double PJ_DMS.d¶

Degrees.

double PJ_DMS.m¶

Minutes

double PJ_DMS.s¶

Seconds.

PJ_EZN¶

Geoid undulation and deflections of the vertical.

typedef struct { double e, z, N; } PJ_EZN
double PJ_EZN.e¶

Deflection of the vertical, eta.

double PJ_EZN.z¶

Deflection of the vertical, zeta

double PJ_EZN.N¶

Geoid undulation.

PJ_AF¶

Ellipsoidal parameters.

typedef struct {double a, f; } PJ_AF
double PJ_AF.a¶

Major axis of ellipsoid.

double PJ_AF.f¶

Flattening of ellipsoid.
